## CCDSALG T3 AY 2025-2026 MCO1
This program is a calculator that reads normal math expressions in infix format and converts it into postfix format, then solves it aftwerwards. It uses a Queue to keep numbers in order and a Stack to sort operators based on its precedence, it uses another stack to read the sorted expression and pop pairs of numbers to calculate intermediate results until only the final answer remains.

# **How to run the program**

Enter this in terminal to compile the program: 
```
javac Converter.java DataSetGenerator.java
```
    
Enter this in terminal to pick and generate a data set:
```
java DataSetGenerator
```   
Enter this in terminal to run the program:
```
java Calculator
```
